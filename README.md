# Hyprland Arch Dotfile

My first attempt at working on my own dotfiles.

This very much still a work in progress.

# Usage

1. Install chezmoi

```bash
sudo pacman -S chezmoi
```

2. Run chezmoi init
```bash
chezmoi init https://github.com/isaac-varg/dotfiles.git
```
